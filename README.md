# Website for Pete's Candy Store, a bar/music venue in Brooklyn, NY.  Made by Jacob Silver

## Technologies

1. **React**

    version 18.2.0. This site makes extensive use of hooks, specifically useState. Mostly functional components with TypeScript support.

2. **Next.js**

    Full-stack React framework with server-side rendering and static generation, pulling data from Airtable.

3. **TypeScript**

    Full TypeScript implementation for type safety and better developer experience.

4. **Airtable**

    Most of the data on this website is stored on Airtable, allowing the staff to change the content of the site without having to know anything about the wild world of JS/HTML/CSS. This site makes extensive use of Airtable's javascript library.
